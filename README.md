# twitter-sentiment-app
Twitter Sentiment Analysis Web App An end-to-end NLP project that classifies tweets into Positive,Negative, or Neutral sentiments. Includes text preprocessing, TF-IDF vectorization, and ML models like Logistic Regression and Naive Bayes. Deployed with Streamlit for real-time predictions.
